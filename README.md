# BangkokExpress - ресторан с онлайн-заказом
Учебный проект по JavaScript.
Сверху карусель с товарами, ниже ленточное меню, слайдер-ползунок, далее подгрузка товаров с сервера, динамическая фильтрация списка товаров. А также корзина товаров с динамическим позиционированием при прокрутке, анимацией, модальным окном и формой заказа.
Используется компонентная архитектура.
